## Key Technologies

- **HTML5** — semantic markup;
- **CSS3** — styling, responsive design, and animations;
- **JavaScript (ES6+)** — application logic and DOM manipulation.

## Libraries & Tools

- **Vite** — project bundling and development server;
- **GSAP** — high-performance professional animations;
- **Axios** — HTTP requests to the API;
- **Swiper.js** — implementation of the reviews slider;
- **iziToast / Notiflix** — stylish pop-up notifications and alerts;
- **SVG Sprite** — efficient icon management;
- **Git / GitHub** — version control and team collaboration;
- **GitHub Pages** — project deployment.

## Project Overview

artistsHUB is a responsive web application that allows users to:

- Explore detailed information about musical artists;
- Read reviews shared by other users;
- Submit their own feedback via a rating-integrated form;
- Seamlessly interact with the interface across mobile, tablet, and desktop
  devices.

**The project was developed as a team collaboration, utilizing modern frontend
development tools and adhering strictly to technical requirements.**
